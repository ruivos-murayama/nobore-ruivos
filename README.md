# のぼれRUIVOSくん (Nobore RUIVOS-kun)

指1本で遊ぶワンタッチ・アクションゲーム。
モバイルゲーム **Ordia**（Loju）の中核メカニクス＝「引っぱって発射」「どんな面にも貼り付く」を
徹底的に再現し、**忍び込み（夜の城・見張りの目・隠れ蓑）** のテイストに作り替えたオリジナル作品です。

> アセット・コードはすべてオリジナル。Ordia の画像／プログラムは一切複製していません。
> 模倣しているのは「操作感（ゲームメカニクス）」のみです。

## あそびかた

1. `index.html` をブラウザで開く（ビルド不要・外部ライブラリなし）。
2. **RUIVOSくんを押して引っぱり、指をはなすと発射。** 引くほど強く飛ぶ（タメ中はスロー）。
3. 壁でも天井でも、触れた面に **ペタッと貼り付く** → そこからまた発射。
4. **見張りの視線**に入ると発見メーターが溜まり、満タンで見つかってやり直し。物陰に隠れ、首振りの隙に跳ぶ。
5. **蛍光色の致死面**に触れる・**画面下に落ちる** とミス（その場で復活）。
6. **光る出口**に入ればステージクリア。

PC ＝マウス、スマホ／タブレット＝タッチ、どちらも同じ操作です。

### 縛り：ジャンプ回数（↗）

ステージごとに弾ける回数に上限があり、使い切って出口に届かないとやり直し。
道中の **💧雫** を拾うと回数が少し回復＝集めるほど長く飛べる（難易度の調整弁）。回数は毎リスポーンで満タンに戻ります。

ギミック：**バンパー**（必ず気持ちよく跳ね返す）／**カタパルト＝射出花**（触れると決まった方向へ一気に撃ち出す。ジグザグに並べた花を1発で連鎖して大跳躍）／**フープ＝輪**（バスケのフリースロー：弧を描いて上から落とし込むと「スウィッシュ！」、決まると上のエリアへワープ。下からは入れない・リムに当たると弾かれて台無し）／**上昇気流**（入ると一気に加速）／**隠れ蓑の雫**（一定時間 透明化）／**コンボ雫**（1回の飛行で連続回収すると音程↑）。

## ファイル構成

```

├── style.css    UI スタイル（ステージ配色を CSS 変数で共有）
├── design.js    物理・手触り・配色の単一の置き場（★まず弄る）

├── game.js      ゲーム本体（物理・ステルス・描画・サウンド）
├── DESIGN.md    デザインシステム解説（どの値をどこで変えるか）
```

（`feel.html` / `feel.js` は手触り検証用サンドボックス。本編とは独立。）

## 実装のポイント（Ordia 再現要素）

- **スリングショット発射**：引いた距離→威力、逆向きへ飛ぶ。点線で軌道プレビュー。
- **全方向スティック**：円×多角形の衝突から法線を求め、床・壁・天井いずれにも貼り付く。
- **固定タイムステップ物理**（120Hz＋描画補間）：フレームレートに依存しない安定挙動。
- **ステルス**：見張りの視線コーンが首を振り、壁で遮られる（物陰に隠れられる）。
- **スクワッシュ＆ストレッチ**・パーティクル・画面シェイクで手触りを強化。
- WebAudio による軽量SFX。

## 調整したいとき

数値はすべて **`design.js`** に集約（くわしくは `DESIGN.md`）。

- `physics.gravity` … 重力の強さ
- `physics.launchMul` / `physics.maxPull` … 発射威力／引っぱり上限
- `gimmick.bumperRestitution` / `gimmick.boostAccel` … バンパーの跳ね／気流の加速
- `rules.launchPerDango` … 💧で回復するジャンプ回数
- `LEVELS[].maxLaunch`（`game.js`）… ステージごとのジャンプ回数上限
- 洞窟の形・難易度 … `cave.js` の `gen`、各ステージの `gen`（`game.js`）
